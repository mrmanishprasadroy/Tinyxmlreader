### Run this command in terminal to start the app 
streamlit run TLGDefinitionReader.py

### Folder structure for the app  is 

        TlgPlotter 
        |
        |XML
            | Tlg XML file 
        |Data
            |  Log Files with data 
        TLGDefinitionReader.py
        
### Dependency PIP Install 
#### use Python > 3
#### pip install streamlit
#### pip install plotly 
#### pip install pandas
#### pip install matplotlib
